# PAUSE - Your Emotional Guide

A privacy-first emotional support tool that helps you understand your feelings without judgment. No data is ever saved or stored.

## Features
- Complete privacy (everything happens on your device)
- No data storage or tracking
- Three response formats: Quick, Key Points, Detailed
- Dark/Light mode
- Works offline
- Responsive design
